# RNTSFire

RNTSFire sample rn application with typescript template.We use firebase server.

  - Typescript project
  - Firebase(Web Connection)
  - React Native 

> npm install --save


### Installation


```sh
$ cd RNTSFire
$ npm install --save
$ react-native run-ios
```
